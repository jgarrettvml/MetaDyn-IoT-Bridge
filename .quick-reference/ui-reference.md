# UI Reference

## Header

- Shows MetaDyn logo, project name, and BLE status.
- Pair button initiates Web Bluetooth pairing.

## Conversation panel

- Empty state: connection awaiting audio.
- Live input transcription: right-aligned blue bubble.
- Live output transcription: left-aligned orange bubble.
- Completed turns are stored in the message list.

## Visualizer

- Animated bars indicate AI output activity.
- Status text reflects hardware readiness and downlink state.

## Status bar

- NODE: BLE connection to ESP32.
- GATEWAY: Gemini Live session state.
